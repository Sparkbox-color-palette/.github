# Sparkbox — Portable Color Palette & Gradient Manager

## About Sparkbox

Sparkbox provides a complete color palette and gradient management environment for UI/UX designers and developers. Sparkbox combines color palette creation, gradient editing, Photoshop/Sketch sync, and code export. Sparkbox enables users to organize colors, generate gradients, and export as CSS, Swift, Android XML, and more. Sparkbox runs on desktop platforms. Sparkbox is trusted by UI designers, web designers, mobile app developers, and brand managers worldwide.

## Deep Dive — Color Management Notes

Color palettes: create, edit, import, organize colors, color harmonies
Gradients: linear, radial, angular, multiple stops, angle control
Sync: Photoshop swatches, Sketch document colors, Illustrator color groups
Export: CSS, Swift (UIColor), SwiftUI (Color), Android XML, JSON, PNG
Formats: HEX, RGB, HSL, CMYK, LAB, HSB, color names
